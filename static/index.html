Prowler Security Assessment API

A powerful API for running Prowler security assessments on AWS environments and generating compliance reports based on AWS Well-Architected Framework Security Pillar.

Features

  • Run comprehensive AWS security assessments
  • Generate detailed compliance reports
  • Map findings to AWS Well-Architected Framework
  • Export results in multiple formats (JSON, Text)
  • Automated security checks for AWS services
  • Detailed remediation recommendations

Available Endpoints

GET /api/run-prowler-generate-checks-report-fundings

Run Prowler security checks and generate findings report

GET /api/export-compliance-report

Generate compliance report mapped to AWS Well-Architected Framework
